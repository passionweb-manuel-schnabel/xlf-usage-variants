# Context Menu in the TYPO3 Backend

## What does it do?

Shows the usage of XLF Translations in different context (e.g. Fluid, Extbase, Non-Extbase and JavaScript)

## Installation

Add via composer:

    composer require "passionweb/xlf_usage_variants"

* Install the extension via composer
* Flush TYPO3 and PHP Cache

## Requirements

This example uses no 3rd party libraries.

## Extension settings

There are no extension settings available.

## Troubleshooting and logging

If something does not work as expected take a look at the log file.
Every problem is logged to the TYPO3 log (normally found in `var/log/typo3_*.log`)
